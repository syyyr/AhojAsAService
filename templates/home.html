AHOJ as a Service

v0.0.1

Úvod

AAAS (AHOJ as a Service) je úplně supr REST API řešení, pokud zrovna potřebujete někomu říct AHOJ.

Typy návratových dat

AAAS vrací data v následujících formátech, které si lze vyžádat pomocí hlavičky Accept:

text/plain Vrátí obsah jako plain text string.
application/json Vrátí obsah jako JSON objekt.
text/html Vrátí obsah jako stránku, co je minimalistická af.
application/xml lol jk

Operace

Všechny requesty jsou pouze typu GET.

Endpojnt Popis
/version Vrátí verzi AAAS
/ahoj Vrací "AHOJ"
/ahoj/:komu/:od Vrací "AHOJ komu --od"
Parametr od může být vynechán.
